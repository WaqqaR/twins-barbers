# Put your photos here

Reference them from `src/config/shop.js`. Suggested files (any that are missing
just show a styled placeholder, so add them whenever you're ready):

| File | Used for | Tip |
|---|---|---|
| `hero.jpg` | Big landing-page background | Wide, high-res (≥ 2000px), darker images look best behind the white text |
| `about.jpg` | Photo beside the "About" section | Portrait-ish (4:5) — the shop floor, a chair, etc. |
| `logo.png` / `logo.svg` | Navbar logo (optional) | Transparent background; leave `images.logo` empty to use the text wordmark |
| `gallery-1.jpg` … `gallery-6.jpg` | Gallery grid | Mix of cuts, beards, the shop. Set `wide: true` on one or two in the config to make them span 2 columns |
| `barber-marco.jpg`, `barber-devon.jpg`, … | Barber cards | Portrait (4:5), shoulders-up |

You can name files anything you like — just match the names in `shop.js`.
Formats: `.jpg`, `.png`, `.webp`, `.svg` all work. Keep file sizes reasonable
(use ~1600–2400px wide JPEGs at ~80% quality) so the site loads fast.
